# AutoComplete Component
A lightweight React-based autocomplete input built with debouncing, caching, and async API handling.

## Features
- Debounced search with configurable delay
- Keyboard navigation (↑ ↓ Enter)
- Cancelable fetch using AbortController
- Caching for previously searched results
- Highlighted matched text for improved UX

## Tech Stack
- React (Vite)
- JavaScript (ES6+)
- CSS (Modules/Tailwind if applicable)
- Fetch API

## Run Locally
git clone https://github.com/<your-username>/autocomplete-component.git
cd autocomplete-component
npm install
npm run dev
